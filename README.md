# Library Management System (Java Swing)

This project is a "Library Management System" developed using "Java and Java Swing".  
It provides a "graphical user interface (GUI)" to manage library operations such as user login, book details viewing, and role-based access for users and admins.


## Login Module
- Displays a "Login Frame" as the entry point of the application
- Allows users to log in based on their role (Admin / User)
- Acts as the main gateway to access other features

## Book Management
- Displays "book details" such as:
  - Book ID
  - Book Name
  - Author
  - Availability status
- Designed to support future operations like issuing and returning books

## User Module
- User-specific interface
- Allows users to view available books and their details
- Read-only access for general users

## Admin Module
- Admin-specific interface
- Designed to manage:
  - Books
  - Users
  - Library records
- Has higher privileges compared to normal users
